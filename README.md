# quinn-platform

Agent-friendly Quinn Harness monorepo.

## Packages

- `@totoday/quinn-sdk` - TypeScript SDK for Quinn Platform API
- `@totoday/quinn-cli` - CLI wrapper for common agent workflows

## Layout

- `packages/sdk`
- `packages/cli`
- `skills/quinn-admin`

## Development

```bash
pnpm install
pnpm -r build
```
